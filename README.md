# Akoin

A responsive Next.js website for the Institute of Digital Risk (IDR).

## Demo

Checkout here : https://chandani-akoin.netlify.app


## Features
- Responsive navbar with mobile toggle button
- Smooth scrolling navigation
- Sticky header
- Modern design with hover effects
- Contact form
- Multiple sections (About, Services, Community, Contact)

## Technologies Used
- Next.js 16.1.6
- React
- TypeScript
- Tailwind CSS
- Vite

## Getting Started
1. Clone the repository 
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open http://localhost:3000 in your browser

## Project Structure
```
akoin/
├── app/
│   ├── page.tsx      # Main page with responsive navbar
│   ├── layout.tsx    # Layout configuration
│   └── globals.css   # Global styles
├── public/           # Static assets
└── package.json      # Dependencies and scripts
```

## Responsive Navbar
The navbar features:
- Desktop: Horizontal menu with hover effects
- Mobile: Hamburger toggle button that opens a slide-out menu
- Cross icon for closing the mobile menu
- Smooth transitions and animations
